üîµ Plaid Integration Flow:

Step 1: Link Token Creation

  1. Backend creates link_token via /link/token/create
  2. Link token is returned to frontend

Step 2: User Authentication

  1. Plaid Link opens with link_token
  2. User selects bank and authenticates
  3. Plaid returns public_token

Step 3: Token Exchange

  1. Backend exchanges public_token for access_token
  2. access_token used for all subsequent API calls
  3. Fetch accounts, balances, transactions
